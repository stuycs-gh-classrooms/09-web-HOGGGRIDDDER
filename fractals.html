Different kinds of Fractals

This webpage will display three kinds of fractals, how they look like and the modifcations that have been made to them.

Koch Curve

Original image:

Modifications:
  • Each line that it makes will be of a different color of the rainbow
  • Each line goes forward 2x but the last line color is always black. In all, it should be a random rainbow color followed by a black line
  • Each line's size is increased by 25%

Sierpinski Triangle

Original image:

Modifications:
  • Each of (depth = 2) triangle will be made up of 2 red triangles and 1 blue triangle at the bottom right corner
  • If the input is depth 1 then the triangle will just be black
  • The 3 triangles inside the (depth = 2) triangle will be smaller with more space in between each other
  • The sierpinski (depth = 2) triangle will have no 3rd side

Tree

Original image:

Modifications:
  • The tree's color is changed to grey
  • The tree tip color is changed to black
  • The tree shoots out random rainbow color laser beams
  • The tree branch gets a width which decreases in size until it reaches the top
  • The color laser beams width is increased by 100%
