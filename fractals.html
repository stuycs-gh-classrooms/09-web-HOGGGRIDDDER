Different kinds of Fractals

This webpage will display three kinds of fractals, how they look like and the modifcations that have been made to them.

Koch Curve

Original image:

Modifications:
  • Each line that it makes will be of a different color of the rainbow
  • Each line goes forward 2x but the last line color is always black. In all, it should be a random rainbow color followed by a black line
  • Each line's size is increased by 25%


  def koch_curve(t, depth, size):
    if (depth == 1):
        randomColor = random.randrange(1,8)
        if (randomColor == 1):
            t.pencolor('red')
        if (randomColor == 2):
            t.pencolor('orange')
        if (randomColor == 3):
            t.pencolor('yellow')
        if (randomColor == 4):
            t.pencolor('green')
        if (randomColor == 5):
            t.pencolor('blue')
        if (randomColor == 6):
            t.pencolor('indigo')
        if (randomColor == 7):
            t.pencolor('purple')
        t.fd(size)
        t.pencolor('black')
        t.fd(size)
    else:
        koch_curve(t, depth-1, size * 1.25)
        t.lt(60)
        koch_curve(t, depth-1, size * 1.25)
        t.rt(120)
        koch_curve(t, depth-1, size * 1.25)
        t.lt(60)
        koch_curve(t, depth-1, size * 1.25)

Sierpinski Triangle

Original image:

Modifications:
  • Each of (depth = 2) triangle will be made up of 2 red triangles and 1 blue triangle at the bottom right corner
  • If the input is depth 1 then the triangle will just be black
  • The 3 triangles inside the (depth = 2) triangle will be smaller with more space in between each other
  • The sierpinski (depth = 2) triangle will have no 3rd side


  def triangle(t, size):
      t.lt(60)
      t.fd(size)
      t.rt(120)
      t.fd(size)
      t.rt(120)
      t.fd(size)
      t.rt(180)

  def sierpinski(t, depth, size, scale_factor=1):
      if depth == 1:
          triangle(t, size/2)
      else:
          t.pencolor('red')
          sierpinski(t, depth-1, size/2)
          t.fd(size/2)
          t.pencolor('blue')
          sierpinski(t, depth-1, size/2)
          t.pencolor('red')
          t.bk(size/2)
          t.lt(60)
          t.fd(size/2)
          t.rt(60)
          sierpinski(t, depth-1, size/2)
          t.rt(120)
          t.fd(size/2)
          t.lt(120)

Tree

Original image:

Modifications:
  • The tree's color is changed to grey
  • The tree tip color is changed to black
  • The tree shoots out random rainbow color laser beams
  • The tree branch gets a width which decreases in size until it reaches the top
  • The color laser beams width is increased by 100%
  • The size is increased by 20%


  def tree(t, depth, size, angle, width):
      t.width(width)
      if depth == 0:
          t.pencolor('black')
          t.fd(size/2)
          t.bk(size/2)
          t.pu()
          t.fd(size)
          t.pd()
          t.width(width * 2)
          randomColor = random.randrange(1,8)
          if (randomColor == 1):
              t.pencolor('red')
          if (randomColor == 2):
              t.pencolor('orange')
          if (randomColor == 3):
              t.pencolor('yellow')
          if (randomColor == 4):
              t.pencolor('green')
          if (randomColor == 5):
              t.pencolor('blue')
          if (randomColor == 6):
              t.pencolor('indigo')
          if (randomColor == 7):
              t.pencolor('purple')
          t.pu()
          t.fd(size)
          t.pd()
          t.fd(size/2)
          t.bk(size/2)
          t.pu()
          t.bk(size)
          randomColor = random.randrange(1,8)
          if (randomColor == 1):
              t.pencolor('red')
          if (randomColor == 2):
              t.pencolor('orange')
          if (randomColor == 3):
              t.pencolor('yellow')
          if (randomColor == 4):
              t.pencolor('green')
          if (randomColor == 5):
              t.pencolor('blue')
          if (randomColor == 6):
              t.pencolor('indigo')
          if (randomColor == 7):
              t.pencolor('purple')
          t.pd()
          t.fd(size/2)
          t.bk(size/2)
          t.pu()
          t.width(width)
          t.bk(size)
          t.pd()
      else:
          t.pencolor("grey")
          t.fd(size)
          t.rt(angle)
          tree(t, depth-1, size * 1.2, angle, width-1)
          t.lt(2 * angle)
          tree(t, depth-1, size * 1.2, angle, width-1)
          t.rt(angle)
          t.pencolor('grey')
          t.bk(size)
